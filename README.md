# BloxterChromeApp
Bloxter Chrome App

## Introductory Goals
1.  Check if you connected to Bloxter.com or a robot.
2.  Check battery level.
3.  Check the name of the robot.
4.  Rename the robot.

## Advanced Goals
1.  Change the name of the robot.
2.  Reboot the robot.
3.  Advanced options.
4.  Download debug scripts.
5.  Download code.
